# Vue Heroku Boilerplate

![logo](https://repository-images.githubusercontent.com/238940698/6cba0b80-49bd-11ea-932b-d0952d3d59e0)

- Vue 2
- Vue-CLI 3
- Node, Express

## Getting Started

Clone project and enter in folder:

```
cd vue-heroku-boilerplate
```

Run npm install:

```
npm i
```
Enjoy:

```
npm run serve
```
